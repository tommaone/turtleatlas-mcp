# Journey Files

Place your detailed process/workflow `.md` files here.

Journey files are deep-detail extracts of specific business processes or system interactions. They are richer and larger than expert files (~10–25k tokens each).

Use journeys when you need:
- Complete business rules for a specific workflow
- Field validations and step-level logic
- State machine transitions with guards and side effects
- Integration contracts between systems

**Load only what is directly relevant** — the LLM context window is finite.

See `CLAUDE.md` for the full guide on how to structure journey files.
