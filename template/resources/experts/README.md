# Expert Knowledge Files

Place your domain expert `.md` files here.

Each file should explain the **business meaning** of a domain area:
- What the key tables are and what they store
- What the important columns mean in business terms
- What domain/enum values mean
- Common query patterns for this domain
- Gotchas and non-obvious behaviours

**What does NOT belong here:**
- Step-by-step SQL procedures
- Long lists of queries covering every scenario
- Column definitions already in the schema archive (those go in `tables.zip`)

See `CLAUDE.md` for the full guide on how to add expert files using the `/add-expert` skill.
